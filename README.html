Jerarquía de clases que permite evaluar una cadena que contenga una expresión del tipo:
    X=5.0+8.3*sinl(M_PI/2.0)


Los fuentes de este proyecto puede descargarse de GitHub mediante
    $ git clone git://github.com/antoniocarrillo69/EvaluaExpresion.git
Para compilar usar:
    $ g++ *.cpp
Ejecutar mediante:
    $ ./a.out
